# Prompt — Gerar Épico (Claude.ai)

> Roda no Claude.ai. Escreve um Épico a partir do template. Cole o bloco abaixo, preenchendo os `[campos]`.

```text
Pela lente de PM, escreva um Épico usando o template docs/templates/epico.md para o release: [RELEASE] — [TÍTULO].
O Épico é um PRD enxuto: carrega só a intenção (problema/motivação, usuário, resultado esperado, objetivo de aprendizado, métricas de sucesso, escopo de alto nível, fora de escopo, premissas/restrições).
Expanda a entrada deste release no roadmap — NÃO duplique o roadmap nem reescreva as features em detalhe.
Escopo em bullets de uma linha; o detalhe de cada Story mora na Spec, não aqui.
Referencie RFCs/ADRs relacionados como restrições já decididas, sem re-justificar.
NÃO liste subtasks nem rastreie status — o Épico não é artefato de board (o agrupamento das Stories é via release + prefixo numérico).
Alinhe comigo o que estiver ambíguo antes de escrever.
```
