# Prompt — Gerar RFC (Claude.ai)

> Roda no Claude.ai. Escreve uma RFC a partir do template. Cole o bloco abaixo, preenchendo os `[campos]`.

```text
Pela lente de Tech Lead, escreva uma RFC usando o template docs/templates/rfc.md para a feature: [FEATURE].
Inclua resumo, motivação, design proposto, alternativas, riscos/NFRs e questões em aberto.
Enxuta por padrão: só o necessário pra revisar e opinar; nada prolixo, referencie em vez de reexplicar. Não corte substância (decisão, alternativas, riscos) — corte padding. Extensão só se o conteúdo exigir.
Ao final, aponte quais ADRs essa RFC deve gerar.
Alinhe comigo o que estiver ambíguo antes de escrever.
```
